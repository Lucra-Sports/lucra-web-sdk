# Listening for "Exit Lucra"

Register a listener for when a user attempts to exit the Lucra experience:

```typescript
import { LucraClient } from 'lucra-web-sdk';

const client = LucraClient.getInstance();

client.on('exitLucra', () => {
  console.log('User attempted to exit Lucra!');
});
```

A good place to register this is inside your `loginSuccess` listener, once the user's Lucra session is fully ready.

To stop listening:

```typescript
const handleExit = () => { /* ... */ };

client.on('exitLucra', handleExit);
// later...
client.off('exitLucra', handleExit);
```
