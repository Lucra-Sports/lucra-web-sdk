# Deeplink Support

#### lucraClient.sendMessage.deepLinkResponse(...)

Responds to a `deepLink` event with a transformed URL. Used when Lucra requests a URL that should be opened in your application, which then navigates back to Lucra at the specified location. For matchup invitation links, prefer using `setMatchupDeepLinkHandler` instead.

**Example:**
```typescript
import { LucraClient } from 'lucra-web-sdk';

const client = LucraClient.getInstance();

client.on('deepLink', ({ url }) => {
  client.sendMessage.deepLinkResponse({
    url: `https://yourapp.com?redirect=${url}`
  });
});
```

### Configuring Matchup Invite URL Generation

When users want to share or invite others to a matchup, configure how invite URLs are generated using `setMatchupDeepLinkHandler`:

```typescript
import { LucraClient } from 'lucra-web-sdk';

const client = LucraClient.getInstance();

client.setMatchupDeepLinkHandler((matchupId) => {
  return Promise.resolve(`${window.location.origin}?matchupId=${matchupId}`);
});
```

**How it works:**

1. **Lucra requests an invite URL** - When a user shares a matchup, Lucra calls your handler with the `matchupId`
2. **Your handler generates a shareable URL** - Create a URL that can be shared via social media, messaging, etc.
3. **Receiver opens the URL** - When someone clicks the link, it opens your application
4. **Your app detects the matchupId** - On launch, check URL parameters and navigate to the matchup:

```typescript
const urlParams = new URLSearchParams(window.location.search);
const matchupId = urlParams.get('matchupId');
const containerElement = document.getElementById('lucra-container');

if (matchupId && containerElement) {
  client.open(containerElement).matchupDetails(matchupId);
}
```
